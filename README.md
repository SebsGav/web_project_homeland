# Tripleten web_project_homeland

# De patria a patria

## Descripción del proyecto

Este es un proyecto web desarrollado como parte del programa de TripleTen. Consiste en una página interactiva ("Landing Page") que presenta historias y paisajes de diferentes lugares del mundo (Kentucky, Burundi, Gales y Ucrania), destacando los orígenes de los colegas en línea de la comunidad tecnológica.

## Funcionalidades

- Diseño responsivo adaptado para resoluciones de escritorio (1280px), tablet (768px) y móvil (320px).
- Organización de contenido mediante Flexbox para un diseño limpio y estructurado.
- Uso de fuentes externas y optimización de imágenes para una mejor experiencia visual.

## Tecnologías y técnicas utilizadas

- **HTML5**: Estructura semántica de la página.
- **CSS3**: Estilos personalizados, incluyendo:
  - Metodología **BEM** (Block Element Modifier) para mantener un código organizado y escalable.
  - **Flexbox** para el posicionamiento y alineación de elementos.
  - **Media Queries** para asegurar la adaptabilidad en distintos dispositivos (Responsive Design).
  - Normalización de estilos mediante `normalize.css`.
- **Google Fonts**: Integración de la fuente "Inter".
- **Git y GitHub**: Control de versiones y despliegue.
